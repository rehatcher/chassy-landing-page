Chassy

ระบบหลังบ้านที่จะเข้ามาช่วยในการติดตามผลการดำเนินธุรกิจของคุณ​ ขอแค่คุณทำธุรกิจเกี่ยวกับรถ ไม่ว่าจะเป็น ล้างรถ หรือ ซ่อมรถ รายใหญ่ หรือ รายย่อย เราสามารถช่วยคุณเก็บข้อมูลต่าง ๆ ที่จำเป็น เพื่อนำมาใช้วิเคราะห์และวางแผนการตลาด ที่ตอบโจทย์ความต้องการของลูกค้า

ติดต่อเรา

บันทึกการให้บริการได้

สามารถให้พนักงานบันทึกการให้บริการลูกค้า เพื่อเก็บรวบรวมข้อมูลที่จำเป็นต่อการนำมาวิเคราะห์ยอดขายได้

เฝ้าร้านจากที่ไหนก็ได้

สามารถติดตามการทำงานและยอดขายแบบเรียลไทม์ ได้ง่าย ๆ ผ่านทางเว็บ และ แอพพลิเคชั่น

วิเคราะห์ยอดขายย้อนหลังได้

สามารถดูข้อมูลการขายย้อนหลังง่าย ๆ ผ่าน Dashboard เพื่อวิเคราะห์และวางแผนการตลาดได้

มีระบบแจ้งเตือนรับรถ

สามารถแจ้งเตือนลูกค้าเมื่อให้บริการเสร็จสิ้นได้อย่างง่ายดายเพียงคลิกเดียวผ่านทาง เว็บ หรือ Application บนมือถือ

Chassy Web and Mobile Application

การใช้งาน

เจ้าของร้าน

  • สร้างบัญชีผู้ใช้ของตน
  • สร้างบัญชีร้านค้า (ได้มากกว่า 1 ร้าน)
  • สร้างบัญชีผู้ใช้ของพนักงานแต่ละร้าน
  • สร้างเมนูของร้าน
  • เลือกจำนวนรถที่ใช้บริการและชำระค่าบริการ
  • ดูข้อมูลผ่านทาง Dashboard บน Web หรือ Application

พนักงาน

  • ลงชื่อเข้าใช้บัญชีที่เจ้าของร้านสร้างให้
  • เปิด QR Line ของ Chassy ให้ลูกค้าเพิ่มเพื่อนเพื่อรับแจ้งเตือน
  • เพิ่มรถของลูกค้า โดยแค่ถ่ายรูปหน้ารถให้ติดป้ายทะเบียนเท่านั้น (ระบบจะทำการอ่านเลขทะเบียนรถจากรูป) จากนั้นเลือกรายการที่ลูกค้าใช้บริการจากเมนูที่เจ้าของร้านกำหนด
  • กดเสร็จสิ้นเมื่อให้บริการเสร็จ เพื่อบันทึกระยะเวลาการทำงาน

ลูกค้า (เจ้าของรถ)

  • เพิ่มเพื่อนกับ Line Chassy
  • กรอกเลขทะเบียนรถที่ต้องการรับแจ้งเตือน
  • รอรับแจ้งเตือนเมื่อพนักงานกดเสร็จสิ้นการให้บริการ

Washever

ช่วยในการเก็บข้อมูลที่จำเป็นต่อการนำมาวิเคราะห์ความต้องการของลูกค้า โดยดูข้อมูลได้ผ่าน Dashboard บน App ในมือถือ สามารถดูข้อมูลรายได้ของทางร้านย้อนหลังได้ด้วย
